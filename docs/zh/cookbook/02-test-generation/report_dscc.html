三段覆盖

分段 生成表格中的 case
正常 "10s"、"2m"、"1h"
边界—零 "0s"、"0m"、"0h"
边界—空白 " 5s "、"\t3m\n"
边界—极大 "18446744073709551615s"（u64::MAX）
错误—空串 ""、" "
错误—未知单位 "5d"、"10ms"
错误—负数 "-5s"、"-10m"
错误—非数字前缀 "abc123s"
错误—仅单位 "s"、"m"、"h"

对照验收标准

  • cargo test 通过。
  • tests/utils_test.rs 存在，表驱动 19 行（要求 ≥ 5）。
  • git diff src/utils.rs 为空（sha 一致）。
