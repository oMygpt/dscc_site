分析任务：社交媒体使用强度与心理健康

数据集位于 /tmp/social_analysis/：

  • social_media_user_behavior.csv — 25000 行用户级数据，约 45 列
  • platform_statistics_2026.csv — 18 个平台的聚合统计

请完成以下分析，并把结果写入 /tmp/social_analysis/report_<YOUR_NAME>.md（<YOUR_NAME> 为 dscc 或 claude）。

必答题（数字须精确到两位小数）

  1. 成瘾分组对比：把 addiction_level_1_to_10 分成三组：

    • Low = 1–3
    • Medium = 4–6
    • High = 7–10

    对每组输出：

    • 用户数和占总体的百分比
    • daily_screen_time_minutes 均值
    • sleep_hours_per_night 均值
    • self_reported_mental_health_effect == "Negative" 的占比

  2. 平台排名：按 primary_platform 取用户数前 5 的平台，给出每个平台的：

    • 平均 addiction_level_1_to_10
    • self_reported_mental_health_effect == "Negative" 占比
    • 平均 daily_screen_time_minutes

  3. 年龄组网暴风险：按 age_group 统计 experienced_cyberbullying == True 的比例，指出风险最高的年龄组。

  4. 一句话结论：基于上述数字，用不超过 80 字总结“使用强度 vs 心理健康“的关系。

输出格式

Markdown 报告，含三个小节（成瘾分组、平台排名、年龄组网暴）+ 一行结论。表格用管道符。 不要额外探索分析，只回答上述四题。
