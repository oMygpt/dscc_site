DSCC Documentation

双语文档：每个英文页面都有结构一致的中文镜像版本，位于 zh/ 目录下。

DSCC 是用安全 Rust 写成的本地 coding-agent CLI。本文档覆盖安装、日常使用、完整命令与工具面，以及通过 agents / skills / plugins 扩展 DSCC 的方法。

文档风格

  • 事实优先：不显然的结论都要给出文件路径或 file:line 引用，读者可复核。
  • 少废话：能用表格就不用散文，句子用祈使式。
  • 双语规范：英文在前，### 中文 小节跟在后面，结构保持一致。代码块、路径、命令不翻译。
  • 命令验证：未在真实设备上验证过的 shell 命令用 # [待验证] 标注；贡献者验证通过后再去掉。
