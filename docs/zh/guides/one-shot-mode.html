单次执行

单次执行模式运行一条 prompt 后退出，用于脚本、CI 和管道集成。入口：dscc prompt <text>（main.rs:318）。

参数

参数 作用 来源
--model <id> 模型 ID，若未设置 DSCC_MODEL 则必填 main.rs
--output-format text|json 输出格式，默认 text，仅非交互生效 main.rs
--allowed-tools <names> 工具白名单，逗号或空格分隔 main.rs:263–277
--permission-mode <mode> read-only | workspace-write | danger-full-access main.rs
--dangerously-skip-permissions 等价于 --permission-mode danger-full-access main.rs:240–242

上游兼容参数

DSCC 保留少量上游 CLI 参数，已有脚本可无改动迁移。

参数 等价 来源
-p <text> prompt <text> main.rs:244–256
--print 强制非交互打印模式 main.rs:258–261

示例

dscc --model claude-sonnet-4-6 prompt "list failing tests" --output-format json

提示

  • --output-format json 仅在 stdout 非 TTY 时生效。
  • --allowed-tools 支持 Read,Grep,Bash 或 Read Grep Bash 两种写法。
  • 权限参数叠加时 --dangerously-skip-permissions 优先于 --permission-mode。
